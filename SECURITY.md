🔐 Security Policy

📌 Supported Versions

Currently, only the latest release of ModdingKingz is actively supported and maintained with security updates.
_______________________
| Version | Supported |
| Latest  | ✅        |
| Older   | ❌        |
_______________________

🚨 Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

- 📩 Contact: info.moddingkingz@gmail.com
- ⏱️ Response time: within 48–72 hours

Please include:

- A clear description of the issue
- Steps to reproduce (if possible)
- Any relevant files or screenshots

We will review all reports and take appropriate action.
Valid vulnerabilities will be fixed as soon as possible.

---

⚠️ Do not publicly disclose vulnerabilities before they are resolved.
